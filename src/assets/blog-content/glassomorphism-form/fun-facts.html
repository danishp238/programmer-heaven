üìú Fact #1: Blur Effects Date Back to the 1800s

The First "Blur" Was an Accident

  • In 1826, Joseph Nicéphore Niépce took the first photograph—but it was blurry due to long exposure.

  • Artists later intentionally blurred backgrounds (bokeh effect) to mimic human eye focus.

Digital Blur Was Born in 1984

  • Apple’s MacPaint (1984) introduced the first digital blur tool.

  • Early web designers faked blur with low-resolution images (before CSS filters existed).

üíª Fact #2: CSS backdrop-filter Almost Didn’t Exist

The 10-Year Browser War

  • Apple proposed backdrop-filter in 2014 for Safari.

  • Google/Firefox blocked it for years over performance concerns.

  • Firefox finally enabled it in 2023—but only behind a flag (layout.css.backdrop-filter.enabled).

‚ö° Fact #4: Blur Can Crash Your GPU

The Dark Side of backdrop-filter

  • Each blur layer doubles rendering work.

  • Nested blurs (e.g., blurring a blurred parent) can freeze mobile devices.

Fix: The "Blur Budget" Rule

‚úî 1–2 blur max per viewport
‚úî Use will-change: transform for GPU optimization
‚úî Disable blur on scroll with @media (prefers-reduced-motion)
